golang.cymru

Go Language User Group Wales

Welcome to the golang.cymru homepage! golang.cymru are an informal, friendly group of Go programming language enthusiasts based in Wales UK.

The group was set up so there could be a Go 1.6 launch party a little closer to home than the nearest which seemed to be London. We didn't get there maybe we could get there before a 1.7 launch party.
